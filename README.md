# align.nvim - teeny tiny align-by-string plugin for NeoVim

![demo][./assets/demo.gif] "Simple of align.nvim"

## Install

### Lazy

```lua
{ "NoSpawnn/align.nvim" }
```

## Commands

- `:Align <string>`
    - Align a visual mode selection to the right-most occurrence of `<string>`
